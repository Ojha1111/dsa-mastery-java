## 📊 Pattern Categories

### 🟢 Easy Patterns (Master these first)
- **Prefix Sum** - Fast range sum queries
- **Kadane's Algorithm** - Maximum subarray sum
- **Cyclic Sort** - Missing/duplicate numbers in O(n)
- **Bit Manipulation** - XOR tricks and bit operations

### 🟡 Medium Patterns
- **Merge Intervals** - Calendar and scheduling problems
- **Top K Elements (Heap)** - Kth largest/frequent elements
- **Greedy** - Local optimal choices
- **Matrix DFS/BFS** - Grid traversal and island problems
- **In-place Linked List Reversal** - Pointer manipulation
- **Topological Sort** - Dependency ordering
- **Union Find** - Connected components

### 🔴 Hard Patterns
- **Trie (Prefix Tree)** - Autocomplete and prefix matching
- **2D Dynamic Programming** - LCS, Edit Distance, grid paths
- **Sliding Window + HashMap** - Advanced window problems
